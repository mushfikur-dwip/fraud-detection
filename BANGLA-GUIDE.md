# নির্দেশিকা - ফ্রড ডিটেকশন প্লাগইন

## বাংলায় সম্পূর্ণ গাইড

### প্লাগইন কি করে?

এই প্লাগইনটি আপনার WooCommerce দোকানকে জালিয়াতি অর্ডার থেকে রক্ষা করে। এটি:

1. **ডুপ্লিকেট ফোন নাম্বার চেক করে** - একই ফোন নাম্বার দিয়ে দিনে নির্দিষ্ট সংখ্যক অর্ডার করতে দেয়
2. **ব্ল্যাকলিস্ট ব্যবস্থাপনা** - সমস্যাজনক ফোন/ইমেইল ব্লক করে
3. **হোয়াইটলিস্ট ব্যবস্থাপনা** - বিশ্বস্ত কাস্টমারদের সীমাবদ্ধতা থেকে মুক্ত রাখে
4. **স্থায়ী ব্লক** - বারবার সমস্যা করা কাস্টমারদের চিরতরে ব্লক করে

### ইনস্টলেশন

1. প্লাগইন ফোল্ডার `/wp-content/plugins/` এ আপলোড করুন
2. WordPress এর 'Plugins' মেনু থেকে Activate করুন
3. WooCommerce > Fraud Detection এ যান
4. আপনার সেটিংস কনফিগার করুন

### প্রথম সেটআপ

#### স্টেপ ১: সাধারণ সেটিংস

- **Enable Fraud Detection**: চালু করুন ✓
- **Daily Order Limit**: দিনে কতটি অর্ডার করতে পারবে সেট করুন (যেমন: 3)
- **Check Phone Numbers**: চালু রাখুন ✓
- **Normalize Phone Numbers**: চালু রাখুন ✓

#### স্টেপ ২: মেসেজ কাস্টমাইজ করুন

- **Block Message**: ব্লক হলে কাস্টমারকে যে মেসেজ দেখাবে
  - উদাহরণ: "আপনার অর্ডারটি নিরাপত্তা কারণে ব্লক করা হয়েছে। সাপোর্টে যোগাযোগ করুন।"
- **Limit Message**: লিমিট শেষ হলে যে মেসেজ দেখাবে
  - উদাহরণ: "আপনি আজকের জন্য সর্বোচ্চ অর্ডার সংখ্যায় পৌঁছেছেন। আগামীকাল আবার চেষ্টা করুন।"

### ব্যবহার গাইড

#### ব্ল্যাকলিস্টে যোগ করা

1. WooCommerce > Fraud Detection > Blacklist ট্যাবে যান
2. "Add to Blacklist" বাটন ক্লিক করুন
3. তথ্য পূরণ করুন:
   - **Type**: Phone / Email / IP নির্বাচন করুন
   - **Value**: ফোন নাম্বার/ইমেইল লিখুন (যেমন: 01712345678)
   - **Permanent Block**: চিরতরে ব্লক করতে চাইলে টিক দিন
   - **Reason**: কেন ব্লক করছেন (যেমন: "জালিয়াতি", "চার্জব্যাক")

#### হোয়াইটলিস্টে যোগ করা

1. Whitelist ট্যাবে যান
2. "Add to Whitelist" ক্লিক করুন
3. ফোন নাম্বার/ইমেইল দিন
4. **Bypass Daily Limit**: টিক দিয়ে রাখুন যদি আনলিমিটেড অর্ডার করতে দিতে চান

#### CSV ইম্পোর্ট

বাল্ক এন্ট্রি যোগ করার জন্য:

**ব্ল্যাকলিস্ট CSV ফরম্যাট:**

```csv
type,value,reason,is_permanent
phone,01712345678,জালিয়াতি,yes
phone,01811111111,সন্দেহজনক,no
email,fraud@test.com,চার্জব্যাক,yes
```

1. এই ফরম্যাটে CSV ফাইল তৈরি করুন
2. Blacklist ট্যাবে "Import CSV" ক্লিক করুন
3. ফাইল আপলোড করুন

### সাধারণ পরিস্থিতি

#### পরিস্থিতি ১: দৈনিক 3টি অর্ডার সীমা

**সেটিংস:**

- Daily Order Limit: 3
- Check Phone Numbers: Yes

**ফলাফল:**

- একই ফোন নাম্বার দিয়ে 1ম, 2য়, 3য় অর্ডার = সফল ✓
- 4র্থ অর্ডার = ব্লক ✗
- পরের দিন আবার 3টি অর্ডার করতে পারবে

#### পরিস্থিতি ২: VIP কাস্টমার

**সমাধান:**

1. কাস্টমারের ফোন নাম্বার হোয়াইটলিস্টে যোগ করুন
2. "Bypass Daily Limit" চালু রাখুন
3. এখন আনলিমিটেড অর্ডার করতে পারবে

#### পরিস্থিতি ৩: জালিয়াত কাস্টমার

**সমাধান:**

1. ফোন নাম্বার ব্ল্যাকলিস্টে যোগ করুন
2. "Permanent Block" টিক দিন
3. কারণ লিখুন: "জালিয়াতি - মিথ্যা তথ্য দিয়ে অর্ডার"
4. এখন এই নাম্বার দিয়ে কখনো অর্ডার করতে পারবে না

### অর্ডার লগ দেখা

Order Logs ট্যাবে আপনি দেখতে পারবেন:

- কোন অর্ডার সফল হয়েছে
- কোন অর্ডার ব্লক হয়েছে এবং কেন
- কাস্টমারের ইমেইল, ফোন, IP অ্যাড্রেস
- তারিখ ও সময়

### ফোন নাম্বার ফরম্যাট

প্লাগইন স্বয়ংক্রিয়ভাবে সব ফরম্যাট হ্যান্ডেল করে:

- `+880 1712345678`
- `880 1712345678`
- `01712345678`
- `01712-345678`
- `(0171) 2345678`

সবগুলো নরমালাইজ হয়ে যায়: `1712345678`

### প্রো টিপস

1. **দৈনিক লিমিট নির্ধারণ:**

   - সাধারণ দোকানের জন্য: 3-5 অর্ডার
   - বড় দোকানের জন্য: 5-10 অর্ডার

2. **হোয়াইটলিস্ট ব্যবহার:**

   - নিয়মিত/বিশ্বস্ত কাস্টমারদের যোগ করুন
   - B2B কাস্টমারদের জন্য আদর্শ

3. **ব্ল্যাকলিস্ট ব্যবহার:**

   - শুধু নিশ্চিত জালিয়াতদের যোগ করুন
   - Permanent Block সাবধানে ব্যবহার করুন
   - কারণ সবসময় লিখে রাখুন

4. **নিয়মিত চেক:**
   - সপ্তাহে একবার Order Logs দেখুন
   - অস্বাভাবিক প্যাটার্ন খুঁজুন
   - প্রয়োজনে সেটিংস আপডেট করুন

#### ইমেইল নোটিফিকেশন

- "Admin Notifications" চালু করুন
- আপনার ইমেইল সেট করুন
